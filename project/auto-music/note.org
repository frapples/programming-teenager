* 重构设计 
写于：<2016-10-12 三>

** 用户功能
   - 播放列表。
   - 按歌手/专辑/歌名查找歌曲，可批量将歌曲加入播放列表。
   - 播放列表可以直接播放。
   - 播放列表可以导出。
   - 播放列表可下载离线歌曲到一个文件夹。
     

** API 层
*** 基本 API 层
    - 包含最基本的能获得的 API。
    - 提供一组测试界面，可以在控制台直接测试这些基本 API，是否失效。
*** 封装 API 层
    - 对上一层的一种包装，使得逻辑更符合需要。

      
**  主要模块
*** 播放列表管理器（对播放列表的抽象）
*** 下载管理器
    ** 
** 辅助模块

    针对于以下命令行的格式：
    auto-music [subcommand1] [subcommand2] [-精简参数 [参数数据] ... ] [--完全参数 [参数数据]...]
